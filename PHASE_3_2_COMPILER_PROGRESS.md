## 🔨 **STATEMENT COMPILATION IMPLEMENTED**

### **1. Variable Declarations (let)**
```azl
# AZL: let x = 42
# Generated Assembly:
mov rax, 42
mov [rbp - 8], rax
```

### **2. Control Flow (if/else)**
```azl
# AZL: if x > 40 { say "Big" } else { say "Small" }
# Generated Assembly:
cmp rax, 40
jz else_label
lea rdi, [string_big]
call printf
jmp end_label
else_label:
lea rdi, [string_small]
call printf
end_label:
```

### **3. Loops (for)**
```azl
# AZL: for let i = 0; i < 5; i++ { say "$i" }
# Generated Assembly:
mov rax, 0
loop_label:
cmp rax, 5
jz end_label
lea rdi, [string_format]
call printf
inc rax
jmp loop_label
end_label:
```

### **4. Function Definitions (fn)**
```azl
# AZL: fn add(a, b) { return a + b }
# Generated Assembly:
add:
push rbp
mov rbp, rsp
mov rax, rdi
add rax, rsi
mov rsp, rbp
pop rbp
ret
```

### **5. Output Statements (say)**
```azl
# AZL: say "Hello, World!"
# Generated Assembly:
lea rdi, [string_hello_world]
call printf
```

## 🏗️ **COMPILATION PIPELINE IMPLEMENTED**

### **4-Stage Compilation Process**
1. **AST Compilation**: Convert AST nodes to assembly code
2. **Assembly Generation**: Generate x86_64 assembly instructions
3. **Code Optimization**: Basic assembly optimization and cleanup
4. **Final Assembly**: Generate complete assembly file with data section

### **Assembly File Structure**
```nasm
; AZL Generated Assembly Code
; Target: x86_64 Linux
; Generated by AZL Compiler written in AZL

section .data
string_hello: db "Hello, World!", 0

section .text
global _start

main:
    push rbp
    mov rbp, rsp
    sub rsp, 16
    
    lea rdi, [string_hello]
    call printf
    
    mov rsp, rbp
    pop rbp
    ret

_start:
    call main
    mov rdi, 0
    mov rax, 60
    syscall
```
