# `project/` — same product, grouped paths

This directory is **part of the AZL/AZME repo**, not a discard pile. Files live here so the **repository root stays small** while **git history is preserved** (`git mv`).

| Path | What |
|------|------|
| `project/repo_root/` | Logs, text captures, JSON, PIDs, and `.azl` files that used to sit loose at the repo root |
| `project/entries/` | Root-level **entry** `.azl`, AZME training/usage **docs**, and **training JSON configs** (`entries/azl`, `entries/docs`, `entries/config`) |
| `project/miscreated_http_colon_path/` | Accidental directory created from a bad URL path (kept so nothing is lost) |

**Canonical language/stdlib implementation** remains under **`azl/`**. **`azme/`** is the AZME subsystem. **`project/`** holds material that is still yours to use — it was never “not the project.”
